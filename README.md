# Installation

igSuck requires:
- python
- colorama
- selenium
- chrome

To install run these comands:
```sh
▶ sudo apt install git
```
```sh
▶ git clone https://github.com/JustFaQ/kwTool.git
```
```sh
▶ cd kwTool
▶ pip install -r requirements.txt
```
